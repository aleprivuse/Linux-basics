## file navigation

1. pwd  = it shows wheare are you
2. ls  = show file and folders in you current view
3. cd .. = go up one level on the familiar tree
4. cd = move in there
5. mkdir = create a file
6. touch = it create and empty file
7. echo = it adds text inside the file that touch created : if you use it whit > its gonna generate new text ; if you use >> its gonna add the new text and keeping the old one
8. cat = show the file and text

### here is a example 

mkdir testybr>
touch test/text.txt<br>
echo "Im learning LINUX" > test/text.txt (you can use the >> if you you want to keep the old text)<br>
cat test/text.txt<br>
